## Root Cause Analysis
1. **Syntax Check**: The current scripts.js passes syntax validation (`node --check`)
2. **Function Exposure**: Both `window.toggleEye` and `window.doLogin` are properly exposed
3. **Likely Issue**: Browser is caching an OLD broken version of scripts.js (v13)

## Proposed Fix

### Option A: Clear Cache and Hard Refresh (Quickest)
1. User should hard refresh: Ctrl+Shift+R or Ctrl+F5
2. Or clear browser cache for the site

1. Change script tag in `public/index.html` to force new version
2. Current: `scripts.js?v=2026-04-20-v9`
3. Change to: `scripts.js?v=2026-04-20-v16` or newer

### Option C: If Still Failing - Revert Completely
1. Find a known working commit and checkout that version of scripts.js
2. Common working commit: `0933770` (coach fix before receipt changes)
